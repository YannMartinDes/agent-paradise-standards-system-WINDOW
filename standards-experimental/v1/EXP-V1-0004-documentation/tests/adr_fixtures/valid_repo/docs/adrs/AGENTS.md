# Architecture Decision Records - Agent Context

When working on files governed by an ADR, always reference the ADR identifier
in a comment block at the top of the file.

**Example:**
```
// Implements ADR-001-initial-architecture
```

Before modifying a file, check if it references an ADR- identifier.
If so, read the corresponding ADR to understand the governing constraints.

See [README.md](README.md) for the full index of ADRs.
